# マイツール — ToDo / メモ / テキストエディター

個人利用向けの軽量ブラウザアプリです。インストール不要・サーバー不要で、HTMLファイルをブラウザで開くだけで動作します。

---

## 機能概要

### ✅ ToDoリスト
- タスクの追加・編集・削除・完了チェック
- 優先度設定（高 / 中 / 低）
- フィルター表示（すべて / 未完了 / 完了済み / 優先度：高）
- 残件数・完了数のリアルタイム表示

### 📝 簡易メモ
- タイトル・カテゴリ・本文でメモを管理
- カテゴリ：コマンド / スクリプト / プロンプト / その他
- ワンクリックでメモ内容をクリップボードにコピー
- カテゴリ別フィルター表示

### 📄 テキストエディター
- ブラウザを閉じても内容が自動保存される簡易エディター
- 読みやすいフォント / 等幅フォントの切り替え
- 文字数カウント
- テキストを `.txt` ファイルとしてダウンロード
- 全文コピー

### 💾 データ管理（エクスポート / インポート）
- 右上の **≡** アイコンからアクセス
- 全データ（ToDo・メモ・テキスト）を JSON ファイルとして書き出し
- JSON ファイルを読み込んでデータを復元
- 別端末への引っ越しやバックアップに利用可能

---

## データの保存について

全データはブラウザの **localStorage** に保存されます。

- データはサーバーに送信されず、ブラウザ内にのみ存在します
- GitHub Pages で公開していても、データが外部に漏れることはありません
- ブラウザのキャッシュ削除や別ブラウザでは、それぞれ別のデータになります

> 別端末・別ブラウザへの移行には、≡メニューの **エクスポート／インポート** を利用してください。

## 使い方

### ローカルで開く

```
index.html と style.css を同じフォルダに置いてブラウザで開く
```

### GitHub Pages で公開する

1. このリポジトリを Fork またはクローン
2. Settings → Pages → Branch を `main` に設定
3. 公開された URL にアクセス
